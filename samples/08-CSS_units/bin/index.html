Cocktail supports all the CSS units (as long as they don't add any) !

Absolute units

I use cm (centimeters)

I use mm (millimeters)

I use pt (points)

I use in (inches)

I use pc (picas) no idea were this one comes from

Relative units

I use em

I use rem

I use ex

I use ch

I use vw (viewport width)

I use vh (viewport height)

I use vmax (the biggest of viewport height or width)

I use vmin (the smallest of viewport height or width)
